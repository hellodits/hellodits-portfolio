## Quick Download dari Simple Icons

Klik link di bawah, lalu:
1. Klik icon yang muncul
2. Klik "Copy SVG"
3. Buat file baru dengan nama yang sesuai
4. Paste SVG code
5. Save di folder `client/public/icons/`
