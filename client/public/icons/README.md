## üìù Cara Download Icon

### **Contoh: MongoDB dari Simple Icons**

1. Buka https://simpleicons.org/
2. Search "MongoDB"
3. Klik icon MongoDB
4. Download SVG
5. Rename menjadi `mongodb.svg`
6. Simpan di `client/public/icons/mongodb.svg`

### **Contoh: React dari DevIcon**

1. Buka https://devicon.dev/
2. Search "React"
3. Copy SVG code atau download
4. Simpan sebagai `react.svg`
5. Letakkan di `client/public/icons/react.svg`

## üé® Alternatif: Gunakan CDN

Jika tidak ingin download, bisa gunakan URL langsung:

```javascript
// Contoh menggunakan CDN
icon: 'https://cdn.jsdelivr.net/gh/devicons/devicon/icons/react/react-original.svg'
```
