## Ukuran Recommended:

- **Profile Photo**: 800x800px atau 1000x1000px (square)
- **Avatar**: 200x200px atau 400x400px (square)
- **Project Cover**: 800x600px (landscape)
- **Project Screenshots**: 1200x800px (landscape)
- **Blog Cover**: 1200x630px (landscape)

## Format:

- JPG untuk foto
- PNG untuk logo/icon dengan transparency
- WebP untuk performa terbaik

## Tips:

1. Compress semua gambar dengan TinyPNG
2. Gunakan nama file lowercase dengan dash (project-name.jpg)
3. Hindari spasi dalam nama file
4. Max file size: 500KB per gambar

## Placeholder Sementara:

Jika belum punya foto, gunakan:
- https://placehold.co/800x600
- https://unsplash.com (foto gratis)
- https://ui-avatars.com/api/?name=Your+Name (avatar generator)
